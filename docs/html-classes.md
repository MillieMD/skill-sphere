# HTML Classes
(Information for converting sample pages to funcitonal PHP)

## General
- **card**: Floating card with rounded corners, largely used for displaying short course summaries e.g. in search, on home page etc.

## Course Page
- **module**: label/drop-down button for each module
- **module-content**: div not nested inside module, opens when "module" is pressed
- **completed**: class identifies that module has been completed by that user, adds a check mark to the accordion menu
- **review**: review container class
- **review-content**: text of review
